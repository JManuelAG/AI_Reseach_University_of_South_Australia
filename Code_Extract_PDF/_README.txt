The following folder contains started code to extract the abstract and details from a pdf

At the same time it includes a code to do topic modelling based on the extractions found on the Abstract.

Topic_Modeling
This code uses topic modeling on the abstract 

PDF_Extract
Initial exploration of how to convert to text a pdf and extract different values from a converted pdf to xml

PDF_Extract_2
This code has the full functionality found on the executable but on a jupyter notebook.

PDF_Extract_EX
This is the executable to extract the information from the code, is executed from the terminal with the program path and the file path
exmple:
PS C:\Folder> python PDF_Extract_EX.py "C:\Folder\Data"

It will output:
- Text extraction
- XML
- Details extraction
